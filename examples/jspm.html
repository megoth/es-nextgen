Setting up jspm.io

Install jspm CLI:

npm install jspm -g

Create your project:

mkdir jspm-demo && cd jspm-demo

Set up jspm:

jspm init

Set up lib/main.js:

var a = 42;
console.log(`test ${a}`);

Set up index.html:

<!doctype html>
  <script src="jspm_packages/system.js"></script>
  <script src="config.js"></script>
  <script>
    System.import('lib/main.js');
  </script>
