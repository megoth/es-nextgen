Setting up webpack

Install webpack CLI:

npm install webpack -g

Create your project and initialize npm:

mkdir webpack-demo && cd webpack-demo

Set up lib/main.js:

var a = 42;
console.log(`test ${a}`);

Compile it:

webpack lib/main.js bundle.js

Set up index.html:

<!doctype html>
  <script src="bundle.js"></script>

For convenience, set up webpack.config.js:

module.exports = {
  entry: './lib/main.js',
  output: {
    path: __dirname,
    filename: 'bundle.js'
  }
};
